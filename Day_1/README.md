# Flask Documentation Downloader (uv)

This project demonstrates how to use **uv** to manage a Python project that
downloads and parses documentation from a GitHub repository.

## Features
- uv-based project setup
- Downloads GitHub repo as ZIP
- Extracts `.md` / `.mdx` files
- Parses frontmatter metadata

## Tech Stack
- Python
- uv
- requests
- python-frontmatter

## Dataset
- Flask GitHub documentation
